# 개발 방향

## 기능 목록

1. 인증 계정
   - 로그인 / 로그아웃
   - 보안 (추후 구현)
2. 운영자 콘솔
3. 조직도
   - 조직도 보이는 구간 구현
4. 메신저
5. 메일 (추후 구현)
6. 전자결재, 전자서명, 보고(리포트)
7. 대시보드
8. 게시판
9. 캘린더/예약
10. 드라이브/자료실
11. 문서관리
12. Works, 보고(리포트)
13. 근태
14. 경리
15. API (추후 구현)
16. 알림 (추후 구현)
17. 관리자 페이지
